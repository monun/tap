### Paper 확장 라이브러리

---

* #### Features
    * 개체 패킷
    * 가상 개체
    * 가상 발사체
    * 개체별 이벤트 리스너
    * YamlConfiguration을 이용한 문자열 템플릿
    * 추가적인 인벤토리 함수
    * GitHub를 통한 업데이트 (BETA)
    * Tick 기반 태스크 스케쥴러 (Ticker)

* #### Environment
    * JDK 16
    * Kotlin 1.5.30
    * Paper & Spigot 1.17, 1.17.1

---

#### Gradle

```kotlin
repositories {

}
```

